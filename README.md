# Sitio Web con JavaScript Básicos

Proyecto desarrollado como práctica de JavaScript, manipulación del DOM y eventos.

## Páginas

- **index.html**      --> Menu de meriendas con lista generada por JavaScript 
- **destacados.html** --> Tarjetas de comidas seleccionables
- **listado.html**    --> Listado de paises con filtrado por continente

## Tecnologias Utilizadas 

- HTML
- CSS
- JavaScript 

## Conceptos

- Modelo de Caja(margin, padding, border)
- Manipulacion del DOM. 
- Eventos de clic.
- Metodos de arreglos(forEach, filter).
- classList(add, toggle).
